# ClipScope v0.1.8 リリースノート
リリース日: 2026-03-26

## 変更概要
ヘッダー表示と設定タブまわりを改善し、配布時のライセンス案内も整備しました。

## 主な改善

- ヘッダーに現在のバージョンを表示するよう変更
- セットアップタブの OBS 連携 URL が、設定したローカルサーバーポートに追従するよう改善
- `THIRD_PARTY_LICENSES.md` の自動生成と配布物への同梱を追加
- 設定タブ下部に作者情報セクションを追加
- ホームページ、X、Twitch、YouTube、BOOTH への favicon リンクを追加

## 互換性・注意点

- favicon 画像は `assets/icons` のローカル同梱ファイルを使用します
- onefile 配布版でも作者リンク用アイコンが表示されるよう、ビルド設定を更新しています
- 既存 DB（`%APPDATA%\\ClipScope\\clipscope.db`）はそのまま利用できます
