# ClipScope v1.0.0 リリースノート
リリース日: 2026-03-31

## 変更概要
ClipScope を正式版 v1.0.0 として公開しました。

## 正式版で提供する内容

- Twitch クリップの監視、一覧表示、削除
- OBS Browser Source へのクリップ再生
- 再生済み / 未再生の管理
- クリップ再生終了後の自動選択解除
- 同一クリップの再クリックによる再生停止
- 日本語 UI、ダークモード、通知音、更新確認

## 補足

- 機能面は v0.1.9 をベースにしています
- 既存 DB（`%APPDATA%\\ClipScope\\clipscope.db`）はそのまま利用できます
- 配布物には `README.md`、`LICENSE`、`THIRD_PARTY_LICENSES.md`、`RELEASE_NOTES.md` を同梱します
