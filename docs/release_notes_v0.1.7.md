# ClipScope v0.1.7 リリースノート
リリース日: 2026-03-25

## 変更概要
起動時に低確率で発生していたフリーズを抑えるため、ネイティブウィンドウの最前面設定の適用タイミングを見直しました。

## 主な改善

- 起動時の `always on top` 設定の重複適用を廃止
- ネイティブウィンドウ生成時の初期値のみで最前面設定を反映するよう変更
- 設定画面からの ON / OFF 切り替え時は、引き続き現在のウィンドウへ反映
- 起動時に一定確率で発生していた「応答なし」フリーズの再発率を低減

## 互換性・注意点

- 最前面設定の機能自体は継続して利用できます
- 起動後に設定画面で最前面 ON / OFF を切り替える動作は従来どおりです
- 既存 DB（`%APPDATA%\\ClipScope\\clipscope.db`）はそのまま利用できます
