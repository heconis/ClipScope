# ClipScope v0.1.2 リリースノート
リリース日: 2026-03-18

## 変更概要
v0.1.1 の安定性を維持しつつ、メイン画面の視認性改善とテーマ対応を中心に UI を強化しました。

## 主な改善

- メインタブのクリップカードを再設計
- サムネイル表示を追加（`thumbnail_url` を利用）
- ステータス / 作成者 / タイトル / 日時 / 削除操作のレイアウトを整理
- 長いタイトルの省略表示（`...`）とホバー時の全文ツールチップ対応
- クリップ日時表示を JST（日本時間）に統一
- 設定タブの一般設定にテーマ切替（ライト / ダーク）を追加
- テーマ変更を保存前プレビュー可能に改善（選択時に即時反映）
- ダークモード時のセットアップ / メイン / 設定の各カード配色を調整
- クリップ一覧更新時の再描画を最適化し、サムネイル点滅を抑制
- 起動時認証の取り扱いを改善
- 起動時にトークン検証を 1 回実行
- 無効トークン時は未認証状態へ自動フォールバック

## 互換性・注意点

- 既存 DB はそのまま利用できます
- OBS には Browser Source としてアプリのローカル再生 URL を設定してください
