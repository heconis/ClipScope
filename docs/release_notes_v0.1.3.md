# ClipScope v0.1.3 リリースノート
リリース日: 2026-03-20

## 変更概要
起動体験とセットアップ導線を改善し、配布版の使い勝手を向上しました。

## 主な改善

- 未認証で起動した場合の認証完了フローを改善
- ブラウザで認証が完了したタイミングで、監視を自動開始するように変更
- 自動開始に失敗した場合はアプリ継続のまま警告を表示し、手動開始へフォールバック
- 配布版（PyInstaller onefile）にスプラッシュスクリーンを導入
- `ClipScope.exe` 起動直後にスプラッシュを表示
- メインウィンドウ表示後にスプラッシュを自動クローズ
- リリースビルドスクリプトを改善
- `build_windows.ps1` で依存インストール / PyInstaller 失敗時に明示的にエラー終了
- ビルド後に `dist/ClipScope.exe` 生成チェックを追加

## 互換性・注意点

- スプラッシュ画像は PNG 形式を使用します
- PyInstaller の既定制約に合わせ、スプラッシュ画像サイズは 760x480 以下を推奨します
- 既存 DB（`%APPDATA%\ClipScope\clipscope.db`）はそのまま利用できます
