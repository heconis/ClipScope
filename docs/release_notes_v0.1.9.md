# ClipScope v0.1.9 リリースノート
リリース日: 2026-03-29

## 変更概要
セットアップタブ・設定タブの見た目を微調整し、クリップ再生まわりの操作感を改善しました。

## 主な改善

- セットアップタブ、設定タブの UI を微調整
- クリップの再生終了後、自動で選択状態を解除するよう改善
- 再生中のクリップを再度クリックしたときに、再生停止できるよう改善
- クリック停止時も通常の再生終了と同様にフェードアウトしてから空画面へ戻るよう改善

## 互換性・注意点

- 既存 DB（`%APPDATA%\\ClipScope\\clipscope.db`）はそのまま利用できます
- `再生済み / 未再生` ステータスは従来どおり保持されます
- 同一クリップの再クリック停止は、選択解除とフェードアウトを伴います
