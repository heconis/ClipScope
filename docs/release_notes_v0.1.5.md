# ClipScope v0.1.5 リリースノート
リリース日: 2026-03-25

## 変更概要
アプリのアップデート確認機能を追加し、設定画面から最新版確認や支援ページへの導線を利用できるようにしました。

## 主な改善

- GitHub 上の `version.json` を参照するアップデート確認機能を追加
- 起動後にバックグラウンドで 1 回だけ最新版を確認する処理を追加
- 設定タブの一般に `自動更新確認` を追加
- 設定タブの一般に `更新を確認` ボタンを追加
- 新しいバージョンが見つかった場合、ダウンロード導線つきの通知ダイアログを表示
- 設定タブの一般に `Ko-fiで支援` ボタンを追加

## 互換性・注意点

- アップデート確認は `version.json` を GitHub 上に公開している前提です
- 自動更新や自動ダウンロードは行いません
- ネットワーク未接続や取得失敗時でもアプリは通常どおり利用できます
- 既存 DB（`%APPDATA%\ClipScope\clipscope.db`）はそのまま利用できます
