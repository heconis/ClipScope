# ClipScope v0.1.6 リリースノート
リリース日: 2026-03-25

## 変更概要
クリップ一覧の視認性を改善し、起動時の初期選択状態を見直しました。

## 主な改善

- クリップ一覧カードに再生時間（秒）を表示
- アプリ終了時に選択中クリップをクリアするよう変更
- 次回起動時は、前回のクリップ選択状態を引き継がず未選択で開始

## 互換性・注意点

- クリップ一覧データや再生済み状態は従来どおり保持されます
- `playback_state` のみ終了時にリセットされます
- 既存 DB（`%APPDATA%\ClipScope\clipscope.db`）はそのまま利用できます
