# ClipScope v0.1.1 リリースノート
リリース日: 2026-03-16

## 変更概要
v0.1.0 の安定化と使い勝手改善を中心に、監視精度・UI・配布まわりを改善しました。

## 主な改善

- 監視の `started_at` を最適化
- 監視開始時刻を下限にしつつ、直近取得クリップの作成時刻に追従
- 数秒の安全マージンを入れて、境界時刻の取りこぼしを抑制
- メインタブのスクロール挙動を改善
- ポーリング更新時にスクロール位置が先頭へ戻る問題を修正
- 設定タブの一般設定を追加
- 常に最前面表示の設定
- 新規クリップ取り込み時の通知音設定
- DB の既定保存先を `%APPDATA%\ClipScope\clipscope.db` に変更
- onefile 配布時でも認証状態・設定が保持されるよう改善
- 通知音の探索先を整理（`%APPDATA%\ClipScope\notify.wav` など）
- リリース手順を v0.1.1 向けに更新

## 互換性・注意点

- 初回利用時は `TWITCH_CLIENT_ID` の設定が必要です
- OBS には Browser Source としてアプリのローカル再生 URL を設定してください
