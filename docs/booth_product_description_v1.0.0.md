# ClipScope Booth 商品説明（v1.0.0）

## タイトル案

ClipScope | Twitchクリップ監視 & OBS再生ツール

## 商品説明本文案

Twitch 配信中に作成されたクリップを監視し、選択したクリップを OBS の Browser Source に流せる Windows 向けツールです。

配信中に「今作られたクリップだけを拾って、手元の一覧から選んで OBS にすぐ出したい」という用途向けに作りました。

最初に Twitch 認証を済ませて、OBS にブラウザソース用 URL を一度設定すれば、あとは ClipScope 側の操作だけでクリップを切り替えられます。

## 主な機能

- Twitch Device Code Flow による認証
- 配信中に作成されたクリップのポーリング監視
- クリップ一覧表示、手動更新、削除
- 未再生 / 再生済みステータス表示
- 選択したクリップを OBS Browser Source へ再生
- 同一クリップの再クリックで再生停止
- 再生終了後の自動選択解除
- 日本語 UI
- ダークモード / ライトモード切り替え
- 通知音再生
- 起動時の更新確認

## 使い方

1. アプリを起動します。
2. セットアップタブで Twitch 認証を完了します。
3. セットアップタブに表示される OBS 用 URL を、OBS の Browser Source に設定します。
4. メインタブで監視を開始します。
5. 配信中に作成されたクリップが一覧へ追加されたら、クリックして再生します。

## 動作環境

- Windows 10 / 11
- OBS Studio
- Twitch アカウント

## 注意事項

- 本ツールは Windows 向けです。
- Twitch 側の仕様変更により、将来的に動作へ影響が出る可能性があります。
- ウイルス対策ソフトの設定によっては、配布版 exe が警告対象になる場合があります。
